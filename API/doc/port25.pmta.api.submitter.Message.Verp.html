Message.Verp Property

Sets or gets whether this message should use the VERP extension.

public bool Verp {get; set;}

Property Value

If you enable VERP, PowerMTA will encode both the originator and recipient addresses in the originator address used for delivering the message. This may make tracking any bounces easier, since you can always derive the original addresses from the address where the bounce is sent.
