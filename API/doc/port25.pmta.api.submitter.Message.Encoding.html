Message.Encoding Property

Sets the kind of encoding to perform on data. The encoding applies to all thereafter added data.

public Encoding Encoding {get; set;}

Property Value

The actual processing performed depends on the encoding selected: SevenBit and EightBit both select the identity encoding, i.e., no transformation is performed on the data, except that any lines terminated with LF alone are converted to CRLF. In the case of 7-bit encoding, it is your responsibility to ensure that all bytes have their high bit off.

Base64 specifies that anydata added shall be encoded on the fly using base-64 encoding. No transformation is performed on the data prior to encoding, making it possible to transmit binary content. Note, however, that to effectively transmit binary data, you must also provide the appropriate MIME headers. In fact, the selected encoding should, in general, agree with the Content-Transfer-Encoding header you specify.

The default encooding (if this method is never invoked) is SevenBit.
